Merhaba bu sayfada Linux kullanıcıları için birkaç makale yayınlayacağım.
İlk önce bir masaüstü ortamı oluşturmak için sizlere birkaç tavsiyede bulunacağım. Fakat bunlar basit ve
yeni başlayanlar için olacak. eğer Linux'u bilgisayarınıza yeni kurduysanız ve bilgisayarınızı
kişiselleştirmek istiyorsanız ilk önce iconlardan başlayalım. https://snwh.org/paper
Buradaki bağlantı iconlarınızı daha iyi ve şık bir hale getirir. Dağtımınıza göre oradaki kodlar yardımıyla
icon temayı indirin. Fakat ondan önce Gnome-Tweaks, Extensions Manager gibi araçları indirmeniz gerekecek.

# Gnome-Tweaks indirme kodu

sudo apt install Gnome-Tweaks

# Extions Manager indirme kodu

sudo apt install extions

Tabi siz uygulama mağzasından manual indirme yapabilirsiniz.
Şimdi ise arka plan işinizi halledelim siz istediğiniz gibi arkaplanlar kulanabilirsiniz.
Fakat bilgisayarınızın daha hızlı kulanılması açısından sizlere bu arkaplanı öneriyorum.

Eğer bu adımların hepsini eksiksiz yaptıysanız doğru sonuç almanız
olasıdır. Ayrıca birazdaha araştırıp insanların neler yaptığına bakabilirsiniz.
